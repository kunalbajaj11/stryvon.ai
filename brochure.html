Printing Instructions

How to print your brochure:

  • Size: 8.5" x 11" (US Letter Portrait) - Standard two-sided flyer/pamphlet
  • Print Settings: Use high-quality paper (100-120gsm recommended)
  • Browser Print: Press Ctrl+P (or Cmd+P on Mac) → Select "More settings" → Set margins to "None" → Choose "Portrait" orientation
  • Professional Printing: Save as PDF and send to a professional printer. They can print both sides on a single sheet.
  • Two-Sided: This is a simple two-page design - print page 1 on the front and page 2 on the back of the same sheet.
  • No Folding Required: This is a flat two-sided flyer/pamphlet design.
  • Customization: You can edit the HTML file to modify content, colors, or layout as needed.
